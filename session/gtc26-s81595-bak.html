模块一：战略跃迁——CUDA 20年，从"训练霸权"到"AI 五层蛋糕"

CUDA 20周年：从 GeForce 到 AI 帝国

革命性计算的二十年征程

黄仁勋在 GTC 2026 上回顾了 NVIDIA 的二十年征程。这不仅仅是 CUDA 的 20 周年，更是从游戏显卡到 AI 计算平台的史诗级转型。

25年前的起点：可编程着色器

25年前，NVIDIA 发明了可编程着色器（Programmable Shader）——世界上第一个可编程加速器。这个"完全不明显的发明"让加速器变得可编程，诞生了像素着色器。这一发明引领 NVIDIA 不断探索，最终在 20 年前创造了 CUDA。

GeForce：最伟大的营销活动

黄仁勋幽默地称 GeForce 是"NVIDIA 最伟大的营销活动"——"我们在你买得起之前就开始吸引未来客户。你的父母付钱，年复一年，直到你成为计算机科学家，成为我们的开发者和客户。"

正是 GeForce 将 CUDA 带到了世界，使 Alex Krizhevsky、Ilya Sutskever、Geoffrey Hinton、Andrew Ng 等人能够发现 GPU 可以加速深度学习，开启了现代 AI 的大爆炸。

三大平台战略

NVIDIA 现在拥有三大平台：

  • CUDA X：计算平台，包含数千个工具和库
  • Systems（系统）：从 DGX 到 AI 超级计算机
  • AI Factories（AI 工厂）：全新的 AI 基础设施平台

关键洞察：NVIDIA 的图表描述了 100% 的 NVIDIA 战略。最难实现的是底部的"安装基础（Installed Base）"——花了 20 年时间才在全球建立起数亿个运行 CUDA 的 GPU 和计算系统。

AI 的五层蛋糕（Five-Layer Cake of AI）

从 FLOPS 到 Token/Watt 的范式转移

黄仁勋提出了"AI 的五层蛋糕"理念，全面确立了 CUDA X、Systems、AI Factories 的核心矩阵。计算的终极度量衡从"FLOPS 数据吞吐"变成了"Token/Watt 生产效率"。

过去：训练霸权时代

过去几年的生成式 AI 狂热中，"模型训练"是绝对的主旋律（"Once upon an AI time training was the paradigm"）。数据中心被视为存储和分发数据的"大型计算机"。

现在：推理驱动世界

训练只是教模型"如何做"，而推理正在驱动整个世界。数据中心彻底演变为"AI Factories（AI 工厂）"：

  • 核心原料：电力
  • 产出物：商业化的 Token
  • 核心竞争力：建厂的标准化程度与能源转化率
"Inference runs the whole world now" — 黄仁勋

推理的四个阶段演进

过去两年发生了根本性转变：

  1. 2023 - ChatGPT（生成式 AI）：AI 不仅能理解，还能生成独特内容
  2. 2024 - o1/o3（推理 AI）：AI 能够反思、思考、规划，将问题分解为可理解的步骤
  3. 2025 - Cloud Code（Agentic AI）：第一个 Agentic 模型，能够读取文件、编码、编译、测试、评估、迭代
  4. 2026 - 物理 AI：AI 能够在物理世界中行动（机器人、自动驾驶）

计算需求爆炸：黄仁勋指出，过去两年计算需求增加了约100万倍。推理的拐点已经到来——"AI 现在必须思考，为了思考，它必须进行推理（inference）。"

模块二：硬件革命——Vera Rubin NVL72 与 Dynamo 1.0 解耦计算

Vera Rubin NVL72：为 Agentic AI 架构的超级计算机

同构 GPU 包打天下的时代终结

NVIDIA 通过引入7 款全新芯片及 Dynamo 1.0 软件，确立了解耦式推理（Disaggregated Inference）。

Vera Rubin NVL72 物理架构

  • 单 Rack 集成：36 个 Vera CPU + 72 个 Rubin GPU
  • 互联技术：第 6 代 NVLink，260 TB/s 全对全带宽
  • 计算性能：3.6 ExaFLOPS
  • 推理性能：50 PFLOPS NVFP4 精度
  • Transformer 引擎：第 3 代，硬件级自适应压缩

NVFP4：精度与效率的革命

第 3 代 Transformer 引擎引入了NVFP4 精度——不仅仅是 FP4 精度，而是全新的张量核心和计算单元。NVIDIA 已证明可以在不损失精度的情况下进行 NVFP4 推理，获得巨大的性能和能效提升。

100% 液冷与制造革新

  • 所有电缆消失，盲插铜缆连接
  • 安装时间从 2 天缩短到 2 小时
  • 使用45°C 热水冷却，减轻数据中心冷却压力
  • 供应链能力：每月可生产多千兆瓦的 AI 工厂

10年40百万倍：从 2016 年的 DGX-1（170 TeraFLOPS）到 2026 年的 Vera Rubin，计算能力增长了4000万倍。

Dynamo 1.0：解耦式推理的操作系统

E/P/D 三阶段物理分离

过去的 LLM 推理中，编码（Encode）、预填充（Prefill）和解码（Decode）都在同一颗芯片上排队。Dynamo 1.0 正式将这三个阶段打散到独立的硬件池：

E/P/D 解耦架构

  • Encode（编码）：处理输入，理解用户意图
  • Prefill（预填充）：准备 KV Cache，计算密集型
  • Decode（解码）：生成 Token，带宽密集型

NIXL：推理传输库

配合解耦计算，NVIDIA 推出了NIXL（NVIDIA Inference Transport Library），专门负责在庞大的计算节点间进行 KV Cache 状态的无阻塞搬运与动态资源分配。

Groq 3 LPU：专用推理加速

NVIDIA 与 Groq 合作，将 Groq 3 LPU（Language Processing Unit）集成到系统中：

  • 确定性数据流处理器：静态编译，编译器调度
  • 大规模 SRAM：专为推理设计的内存架构
  • Token 加速器：与 Vera Rubin 紧密耦合

商业价值：Vera Rubin 训练万亿参数模型的成本降至上代 1/4，推理成本降至 1/10，兑现了黄仁勋"35倍吞吐量提升"的承诺。

Token Factory：AI 工厂的经济学

Token/Watt 定义新商业模式

黄仁勋展示了关键的Token 工厂效率图表——这是未来每个 CEO 都会深入研究的图表：

双轴分析

  • Y 轴（吞吐量）：每瓦特产生的 Token 数量
  • X 轴（Token 速度）：推理速度，即 AI 的智能程度

分层定价模型

Token 将像所有商品一样分层：

  • 免费层：高吞吐量、低速度
  • 标准层：中等模型、较高速度
  • 高级层：更大模型、更高速度、更长上下文
  • Premium 层：极高速度、关键路径应用（可达 $150/百万 Token）

收入倍增：假设 25% 电力分配给每个层级，Vera Rubin 可产生 5倍于 Blackwell 的收入。如果加入 Groq，可进一步扩展到 1000+ Token/秒的超高端市场。

模块三：软件重构——OpenShell 与 NemoClaw 掌控 GaaS 入口

OpenClaw：人类历史上最受欢迎的开源项目

Agentic 系统的 Linux 时刻

黄仁勋宣布：OpenClaw 是人类历史上最受欢迎的开源项目，在短短几周内就超越了 Linux 30 年的成就。

什么是 OpenClaw？

OpenClaw 本质上是一个Agentic 操作系统：

  • 资源管理：访问工具、文件系统、大语言模型
  • 调度能力：Cron 作业、任务分解
  • 子代理：可以生成和调用其他子代理
  • I/O：支持任何模态的输入输出（语音、文字、手势）
"OpenClaw 开源了 Agentic 计算机的操作系统。正如 Windows 让我们能够创建个人电脑，OpenClaw 让我们能够创建个人代理。"

企业 IT 的重构

OpenClaw 前：企业 IT 是数据存储 + 软件工具 + 人类使用

OpenClaw 后：每个 SaaS 公司都将成为 GaaS（Generative as a Service）公司

每个公司的战略问题："你的 OpenClaw 战略是什么？"正如我们曾经需要 Linux 战略、HTTP/HTML 战略、Kubernetes 战略一样。

OpenShell & NemoClaw：企业级安全护栏

让 Agentic 系统企业就绪

Agentic 系统在企业网络中面临严峻安全挑战：可以访问敏感信息、执行代码、外部通信。NVIDIA 与 Peter（OpenClaw 作者）合作，创建了OpenShell——让企业级 OpenClaw 安全且私密。

NemoClaw 技术栈

  • OpenShell：集成到 OpenClaw 的企业安全技术
  • 策略引擎：连接企业现有的策略系统
  • 网络护栏：控制 Agent 的网络访问
  • 隐私路由器：保护数据隐私

NVIDIA 开放模型计划

  • Nemotron：语言、视觉、RAG、安全、语音

DSX 核心组件

DSX Max-Q：释放隐藏算力

⭐ 洞察：给华为专家的建议

战略反制与对标建议

芯片硬件层-软件调度层-安全运行环境-能源基建层全栈布局

NVIDIA 的这一轮发布，标志着从"芯片硬件层-软件调度层-安全运行环境-能源基建层"形成了极其恐怖的闭环垄断。针对华为各业务线，提出以下战略反制与对标建议：

1. 计算领域 (华为计算/昇腾)

坚定走向"解耦式异构推理池"

NVIDIA Dynamo 1.0 的 E/P/D（编码/预填充/解码）物理分离已成为行业标杆。昇腾团队应立即停止用单一通用算力卡跑全流程大模型推理的做法。建议拆分现有的昇腾集群：

  • 部分 910 节点专攻海量并行计算的 Prefill
  • 研发具有极大容量 SRAM（对标 Groq）的专用推理加速卡，专门负责 Memory-bound 的 Decode 阶段
高速缓存传输库（对标 NIXL）开发

解耦推理的核心难点在于海量 KV Cache 的跨节点转移。星河网络团队需联合昇腾底层，开发类似于 NIXL 的超低延迟显存直接传输协议，用华为在光通信/以太网的积累，弥补 NVLink 的差距。

算-网-电-冷一体化基建（对标 DSX）

发挥华为数字能源（Digital Power）的绝对长板优势。不仅卖算力，更要卖"智能电网+液冷 AI 工厂"的统包方案。在调度层面，实现任务计算峰值与绿色能源（光伏/风电）波谷的 AI 协同，真正把"Token/Watt"做到全球最优。

2. AI/软件领域 (华为云/盘古大模型)

构建自主可控的智能体沙箱（对标 OpenShell）

NVIDIA 试图用 OpenShell 掌控 Agent 时代的系统调用权。华为云（甚至鸿蒙内核）必须立即推出国产化的 Agent 运行环境/沙箱容器（如盘古智能体引擎）。做到"一键部署、强制鉴权、数据不出域"。在政务云、国资央企场景中，自主可控的 Agent 护栏是击败一切国外开源框架的底线要求。

拥抱并魔改开源框架

针对 vLLM、SGLang 等主流开源推理框架，CANN 及 MindSpore 团队应提前植入底层算子支持，确保华为算力能够无缝承接类似 Dynamo 架构的分布式推理需求。

3. 芯片领域 (海思半导体)

应对 NVFP4 精度的生态封锁

Rubin 架构的第 3 代 Transformer 引擎通过硬件压缩支持了 NVFP4。海思在下一代达芬奇架构设计中，必须在极低比特（4-bit 甚至更低）量化计算上实现原生硬件级加速，并保证微架构上的兼容性或自适应能力。

合成数据加速器的前瞻布局

预见到未来"大量算力用于自我生成合成数据"（自我对弈、蒙特卡洛树搜索等），下一代 NPU 的设计应适当加强对数据清洗、校验和快速上下文重建的专门加速模块。

4. 终端/系统领域 (华为车BU/终端业务)

系统级智能体底座的抢位

手机与 PC 端侧，谁能成为最佳的"Agent 宿主（Host）"，谁就能统治下一个十年。鸿蒙系统（HarmonyOS）不仅要开放 API，更要在系统底层建立一套"原生 Agent 安全通信与跨应用执行引擎"（对标 NemoClaw），让端侧设备具备高信任度的自主编排能力。

Alpamayo 的全面阻击

华为车 BU 的云端仿真系统应提升对"3D 神经渲染与合成物理数据"的生成权重。端侧智驾芯片应全面拥抱大语言/视觉模型的 Native 执行，实现端到端的 Sim-to-Streets（仿真即现实）闭环部署。

📈 技术发展趋势分析

未来 3-5 年关键发展方向

2026-2027（架构解耦与沙箱期）

同构 GPU 算力集群开始瓦解。数据中心明确划分为"重算力预填充集群"和"重内存解码集群"。同时，以 OpenShell 为代表的中间件层（Agent 沙箱）将爆发，成为各类大模型与企业数据之间的"安检站"。

2028-2029（能源算力强绑定）

能效比的竞争将白热化。单一企业甚至可能无法独立承担兆瓦级机架的电力成本。"电力调峰与算力调度联合寻优"的技术（如 DSX Flex）将成为云厂商的核心资产。

2029-2030（物理 AI 落地期）

利用庞大算力生成出近乎完美物理规律的合成数据，通用机器人（具身智能）将突破 Corner Case 瓶颈，从封闭试验场走向开放工业/商业环境进行规模化替代。

关键技术路线演进推演

计算架构

算力模块细分，预填充、解码、调度控制彻底物理分离。晶圆级封装 (Wafer-Scale) 与 CPO (共封装光学) 将大规模应用，打破芯片间通信带宽瓶颈。

网络与存储

传统的以太网/TCP-IP 协议在 AI 节点内部完全被摒弃。KV Cache 专用的极低延迟显存网络（类似于内存语义互联）将主导超算通讯。

软件栈

从提供 API 变成提供"独立思考空间的运行容器（Containerized Agents）"。代码编程将部分退化，开发者主要工作变为"配置智能体的护栏策略和目标函数"。

中国与全球市场差异化机会

中国市场的独特挑战与解法

受制于先进制程与顶级互联带宽（如 NVLink 级单节点超大带宽网络），中国市场必须发展出全球最强的"弱单体、慢网络下的极大规模分布式调度算法"。欧美走向"单体无脑暴力堆叠"，中国则需要在"集群精细化容错与算力解耦"上点满科技树。

"基建狂魔"的终极红利

NVIDIA DSX 面临的最大痛点是欧美落后的电网改造速度（导致 100GW 能源受困）。这正是中国的绝对主场。凭借全球第一的特高压电网、绿电并网调度能力以及液冷基础设施土建能力，中国极有潜力在全球率先跑通"特高压微电网 + 纯液冷 AI 巨型工厂"的超级基建模式，从而在"Token/Watt"这一终极核心商业指标上，实现对硅谷的弯道超车。
